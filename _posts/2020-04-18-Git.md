# 常用的上传命令

```
git add .
git commit -m "hong"
git push
```

# 用远程代码强制覆盖本地代码

```
git强制覆盖：
    git fetch --all
    git reset --hard origin/master
    git pull
git强制覆盖本地命令（单条执行）：
    git fetch --all && git reset --hard origin/master && git pull
```
